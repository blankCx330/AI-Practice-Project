## 1. 游戏简介

**Neon Snake（霓虹贪吃蛇）** 是一款现代化设计的贪吃蛇游戏，结合了经典玩法与创新技能系统。游戏采用霓虹赛博朋克风格视觉效果，支持单人练习和多人在线对战两种模式。

### 🎮 游戏特色

- **双重模式**：单机练习/多人联机
- **四大技能**：核弹、冰霜领域、镜像分身、黑洞
- **粒子特效**：丰富的视觉反馈
- **流畅动画**：60FPS渲染，丝滑体验

## 3. 快速开始

### 3.1 启动单人模式（推荐新手）

**步骤 1：** 直接用浏览器打开 `snake.html` 文件

> 💡 **提示**：双击文件或拖拽到浏览器窗口均可

**步骤 2：** 点击主界面「🎮 单人模式」按钮

**步骤 3：** 使用方向键控制蛇移动，开始游戏！

---

### 3.2 启动联机模式

**步骤 1：** 启动游戏服务器

```bash
# 打开终端/命令行
cd 贪吃蛇目录
node server.js
```

**步骤 2：** 浏览器打开 `snake.html`

**步骤 3：** 点击「🌐 在线对战」

**步骤 4：** 输入服务器地址（默认 `localhost:8080`）

**步骤 5：** 点击「⚔️ 加入游戏」

**步骤 6：** 等待其他玩家加入后，点击「开始游戏」

## 5. 操作指南

### 5.1 移动控制

| 按键 | 功能 | 说明 |
|------|------|------|
| ↑ | 向上移动 | 只能90度转向 |
| ↓ | 向下移动 | 不能直接掉头 |
| ← | 向左移动 | 不能直接掉头 |
| → | 向右移动 | 不能直接掉头 |

> ⚠️ **重要**：蛇头朝右时，按左键无效；其他方向同理（防止直接掉头导致自杀）

---

### 5.2 技能按键

| 按键 | 技能名称 | 冷却时间 |
|------|----------|----------|
| Q | 核弹 | 8秒 |
| W | 时间静止/冰霜领域 | 15秒 |
| E | 镜像分身 | 3秒 |
| R | 黑洞 | 25秒 |

## 6. 技能系统

### 6.1 Q技能 - 核弹 💥

**效果：** 向当前方向发射核弹，爆炸后清理全图所有得分点

**详细机制：**
- 发射后沿蛇头方向飞行
- 到达目标位置或边界时爆炸
- 爆炸清除所有现有食物
- 蛇身长度增加（每清除1个食物+1节）
- 获得已清除食物分数 × 15

**使用技巧：**
- 食物多时使用收益最高
- 可以配合蛇身长度使用
- 预判爆炸位置

---

### 6.2 W技能 - 时间静止 ❄️

**效果：** 冻结全场食物8秒

**详细机制：**
- 激活后所有食物暂时无法被食用
- 食物显示特殊冻结特效
- 8秒后自动解冻
- 蛇仍可正常移动

**附加效果 - 冰霜领域：**
- 在蛇头周围创建冰霜区域
- 进入区域的食物被冰冻3秒
- 冰冻食物无法被他人吃掉

**使用技巧：**
- 阻止对手吃食物
- 给自己争取时间
- 配合分身使用

---

### 6.3 E技能 - 镜像分身 👥

**效果：** 创建最多5个AI控制的分身

**详细机制：**
- 每个分身自动寻找最近食物
- 分身吃到食物算入本体分数
- 分身长度加到本体蛇身上
- 分身持续15秒后消失
- 分身不参与碰撞判定

**AI行为：**
- 自动追踪最近食物
- 躲避墙壁和蛇身
- 多个分身会互相避让

**使用技巧：**
- 早期使用建立优势
- 可以获得额外得分
- 创造"幽灵模式"（有分身时可穿墙）

---

### 6.4 R技能 - 黑洞 🕳️

**效果：** 创建黑洞吸引全图食物

**详细机制：**
- 黑洞出现在蛇头位置
- 半径逐渐扩大
- 范围内的食物向蛇头移动
- 被吸入的食物直接获得（+20分）
- 持续约3秒后消失

**使用技巧：**
- 食物密集时使用
- 位置好的话一次可吸多个
- 配合蛇头位置

## 7. 联机指南

### 7.1 本地联机

适用于同一台电脑或局域网内的多人游戏

```bash
# 启动服务器
node server.js
```

其他玩家连接：`ws://你的IP地址:8080`

---

### 7.2 局域网联机

**步骤 1：** 查看本机IP

```bash
# Windows
ipconfig

# Mac/Linux
ifconfig
```

找到类似 `192.168.x.x` 的地址

**步骤 2：** 告诉其他玩家你的IP地址

**步骤 3：** 其他玩家连接 `192.168.x.x:8080`

---

### 7.3 外网联机（与QQ好友）

需要将服务器暴露到互联网，有以下方法：

#### 方法1：ngrok（推荐 ⭐）

```bash
# 1. 下载 ngrok
https://ngrok.com/download

# 2. 启动隧道
ngrok http 8080

# 3. ngrok会给你一个地址，如：
# https://xxxx-xxx.ngrok.io

# 4. 分享给好友
```

#### 方法2：FRP内网穿透

需要公网服务器，配置较为复杂，适合技术人员

#### 方法3：花生壳/NATAPP

图形化界面，操作简单

## 9. 故障排除

### ❌ 游戏无法启动

**问题：** 打开HTML文件后一片空白

**解决：**
1. 检查浏览器是否支持Canvas
2. 尝试Chrome/Edge/Firefox最新版本
3. 清除浏览器缓存后刷新

---

### ❌ 单人模式无法开始

**问题：** 点击单人模式后没有反应

**解决：**
1. 按F12打开开发者工具
2. 查看Console是否有错误
3. 刷新页面重试

---

### ❌ 无法连接服务器

**问题：** 联机模式提示连接失败

**解决：**
1. 确认服务器已启动（终端显示"Server Started"）
2. 检查地址是否正确
3. 检查防火墙是否阻止
4. 尝试localhost而非IP地址

---

### ❌ 游戏运行卡顿

**问题：** 画面不流畅

**解决：**
1. 关闭其他占用高的程序
2. 尝试更快的浏览器
3. 检查是否是性能问题

---

### ❌ 蛇穿墙问题

**问题：** 蛇穿过边界

**原因：** 正常使用镜像分身技能时的正常效果

**说明：**
- 有分身时：可穿墙（幽灵模式）
- 无分身时：撞墙即死

## 10. 游戏规则

### 10.1 得分规则

| 动作 | 得分 |
|------|------|
| 吃普通食物 | +10分 |
| 核弹爆炸 | 食物数 × 15分 |
| 黑洞吸入 | +20分 |
| 分身吃到 | +10分 |

---

### 10.2 死亡条件

**单人模式：**
- 撞墙（无分身时）
- 撞到自身

**联机模式：**
- 撞墙（无分身时）
- 撞到自身
- 被其他玩家撞到

---

### 10.3 胜利条件

- 单人模式：无尽头，获得最高分即可
- 联机模式：存活到最后一人
