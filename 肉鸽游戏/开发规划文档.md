## 一、项目概述

### 1.1 项目目标

**项目名称**: 肉鸽游戏 (Rogue Survivor)  
**项目类型**: 动作冒险 / 街机风格肉鸽游戏  
**核心玩法**: 在有限空间内不断击败敌人获取经验升级，收集金币购买武器和被动技能，挑战更高分数

### 1.2 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端框架 | React | 19.x |
| 语言 | TypeScript | 5.x |
| 状态管理 | Zustand | 5.x |
| 构建工具 | Vite | 6.x |
| 游戏渲染 | Canvas 2D | 原生API |

**需要修改**: 近战武器改为飞行动画

### 5.2 武器飞行系统 ❌ 待实现

**设计说明**: 近战武器攻击方式改变

```
攻击流程:
1. 玩家触发攻击
2. 武器从环绕位置飞向目标敌人（outgoing阶段）
3. 飞行过程中碰撞检测，对敌人造成伤害
4. 穿透次数用完或到达目标位置后进入returning阶段
5. 武器飞回玩家身边（returning阶段）
6. 回到玩家身边后恢复环绕状态（orbit阶段）
```

**实现要点**:
- 武器位置状态管理
- 飞行轨迹计算
- 碰撞检测优化
- 穿透敌人记录

### 5.3 敌人AI追踪 ✅ 已实现

**文件**: `src/game/store.ts`

**现状**: 敌人正常追踪玩家

### 5.4 敌人死亡动画 ✅ 已实现
